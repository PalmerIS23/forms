+-------------+      +-----------------------+

plaintext

|             |     |                |     |               |

Запрос на создание/чтение/обновление/удаление записи

Экспортированные/импортированные данные

Сообщения об ошибках и подтверждения

Ключевые бизнес-процессы:

Основной процесс управления записями:

Отображение списка записей

Добавление новой записи

Редактирование существующей записи

Удаление записи с подтверждением

Обработка изображений

Сохранение в IndexedDB

Экспорт данных в JSON-формат

Импорт данных с заменой существующих данных

Отображение результатов

Отображение понятных сообщений об ошибках
